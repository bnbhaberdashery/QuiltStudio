# BNB QuiltStudio v0.2

Version 0.2 is the first proper fabric-rendering release.

## Included

- React + Vite app
- GitHub Pages deployment workflow
- Modular project structure
- Fabric upload workflow
- Lightweight repeat detection
- Natural repeat tiling
- Randomized block offsets
- Improved realistic quilt rendering
- Quilting overlays
- Repeat preview panel
- PNG export
- Basic yardage estimate

## Deploy on GitHub Pages

In GitHub:

1. Open the repository.
2. Go to **Settings → Pages**.
3. Set **Source** to **GitHub Actions**.
4. Commit this project to `main`.
5. GitHub will build and publish automatically.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
