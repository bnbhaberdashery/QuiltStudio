# QuiltStudio Roadmap

## Version 0.1 — Project foundation
- Modular React/Vite project.
- GitHub Pages workflow.
- Fabric upload workflow.
- Pattern registry.
- Canvas renderer shell.
- PNG export.
- Yardage calculator placeholder.

## Version 0.2 — Fabric repeat detection
- Lightweight repeat analysis for uploaded images.
- Crop-size scoring by edge similarity.
- Softened mirror repeat tile.
- Randomized block offsets.
- Repeat preview panel.
- Improved realistic texture and quilting overlays.

## Version 0.3 — Manual repeat controls
- User-adjustable repeat crop.
- Rotate/scale fabric.
- Lock repeat per fabric.
- Higher quality print export.

## Version 0.4 — Pattern expansion
- Log Cabin.
- Rail Fence.
- Churn Dash.
- Ohio Star.
- Courthouse Steps.

## Version 1.0 — Working studio
- Customer-ready design flow.
- PNG and PDF export.
- Yardage calculations.
- Pattern sheet summary.
